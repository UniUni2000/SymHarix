这版骨架的核心判断

1. 首页不再解释所有内部状态，而是先说 现在该处理哪张单。
2. `Create Issue` 降级成主 CTA / modal，不再占住整块左栏。
3. governance / evidence / architecture 先做摘要，再提供下层明细，不再让用户直接吞字段墙。
4. timeline/history 仍保留，但降为 signal rail，不再和主操作区抢第一优先级。
